Jira to Markdown

Convert Jira Server wiki markup to standard Markdown. Paste Jira markup on the left and see Markdown output on the right.

Syntax Reference

Type Jira Markdown
Headings h1. Title to h6. Title # Title to ###### Title
Bold *bold text* **bold text**
Italic _italic text_ *italic text*
Strikethrough -strikethrough- ~~strikethrough~~
Monospace {{code}} `code`
Links [text|http://url] [text](http://url)
Images !image.png! ![](image.png)
Bullet List * item - item
Numbered List # item 1. item
Code Block {code:lang}...{code} ```lang...```
Quote {quote}...{quote} > ...
Table Header ||h1||h2|| | h1 | h2 |
Table Row |c1|c2| | c1 | c2 |
Horizontal Rule ---- ---
