Bidirectional converter between Jira Server wiki markup and Markdown. Edit either pane and the other updates in real-time. Supports headings, bold, italic, strikethrough, underline, monospace, code blocks with language syntax (`{code:javascript}` / ` ```javascript `), nested lists, links, images, tables, blockquotes, panels, and user mentions. Based on patterns from [J2M](https://github.com/kylefarris/J2M) and the [Confluence wiki markup spec](https://support.atlassian.com/confluence-cloud/docs/insert-confluence-wiki-markup/).
