# Tools Repository

See [AGENTS.md](AGENTS.md) for detailed instructions on creating and maintaining tools.

## Quick Reference

- **One tool = one HTML file** with inline CSS/JS
- **No React** or build-step frameworks
- **CDN dependencies** only (jsdelivr, cdnjs)
- **Mobile-first** responsive design
- Create `{tool-name}.docs.md` alongside each tool
- Update `README.md` when adding tools
